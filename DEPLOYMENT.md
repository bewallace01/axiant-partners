# How to Make Your Website Shareable

This guide will help you deploy your Axiant Partners website so it's accessible online and shareable via a URL.

## Quick Deployment Options

### Option 1: Netlify (Recommended - Free & Easy)
1. Go to [netlify.com](https://www.netlify.com) and sign up for a free account
2. Drag and drop your entire project folder onto the Netlify dashboard
3. Your site will be live instantly with a URL like: `your-site-name.netlify.app`
4. You can customize the domain name in settings

### Option 2: Vercel (Free & Fast)
1. Go to [vercel.com](https://vercel.com) and sign up
2. Click "New Project" and import your project folder
3. Deploy - your site will be live in seconds
4. Get a URL like: `your-site-name.vercel.app`

### Option 3: GitHub Pages (Free)
1. Create a GitHub account at [github.com](https://github.com)
2. Create a new repository
3. Upload all your files to the repository
4. Go to Settings > Pages
5. Select your main branch and save
6. Your site will be at: `your-username.github.io/repository-name`

### Option 4: Traditional Web Hosting
Popular hosting providers:
- **Bluehost** - www.bluehost.com
- **HostGator** - www.hostgator.com
- **SiteGround** - www.siteground.com
- **GoDaddy** - www.godaddy.com

Steps:
1. Sign up for a hosting plan
2. Use their file manager or FTP to upload all files
3. Your site will be at: `yourdomain.com`

## Before Deploying - Update These:

### 1. Update Meta Tags
In `index.html`, update the Open Graph URLs:
- Change `https://axiantpartners.com/` to your actual domain
- Update the image URL to your full domain path

### 2. Update All Internal Links
Make sure all links work correctly:
- Navigation links
- Footer links
- Form action URLs (if you add backend later)

### 3. Test Locally
1. Open `index.html` in a browser
2. Test all pages and links
3. Test the form submission
4. Check mobile responsiveness

## Files to Upload

Upload ALL these files to your hosting:
- `index.html`
- `match.html`
- `faq.html`
- `contact.html`
- `privacy-policy.html`
- `terms-and-conditions.html`
- `styles.css`
- `script.js`
- `logo.jpg`
- `Axiant_PrivacyPolicy_UPDATED.pdf` (if still needed)
- `Axiant_TermsAndConditions_UPDATED.pdf` (if still needed)

## After Deployment

1. **Test Your Live Site**
   - Visit your URL
   - Test all pages
   - Test form submission
   - Test on mobile

2. **Share Your Website**
   - Use the social sharing buttons on the homepage
   - Share the URL directly
   - Add to email signatures
   - Share on social media

3. **Optional: Custom Domain**
   - Most hosting providers let you connect a custom domain
   - Purchase a domain (e.g., axiantpartners.com)
   - Connect it in your hosting settings
